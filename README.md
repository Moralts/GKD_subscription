# @Moralts/GKD_subscription

由 Moralts 长期维护管理的 GKD 订阅规则

## 提前声明

- **禁止在国内平台传播**
- **本仓库仅供本人学习用途**

## 如何订阅？

**我们暂时只提供 Github Raw 的原生文件订阅，未来会提供更多渠道**

复制下面任意的链接导入到 GKD 即可：

- ### Github Raw

```
https://raw.githubusercontent.com/Moralts/GKD_subscription/main/dist/gkd.json5
```

## 项目说明

- **立志为 GKD 伙伴们提供最全，最快，最精准的规则**
- **默认提供开屏广告跳过，其余需自行配置**

## 感谢项目
